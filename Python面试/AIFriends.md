 # ***\*AI伙伴\****

  ## ***\*项目简介\****

  本项目是一个集成了先进人工智能技术的智能对话伙伴平台。它旨在为用户提供一个高度拟人化且具备个性化记忆能力的虚拟交流伴侣。通过融合后端强大的自然语言处理、**智能体（Agent）技术**和前端流畅的用户体验，打造一个既有趣又实用的AI互动空间。

------

  ## ***\*技术架构\****

  本项目采用现代化的前后端分离架构，核心技术栈如下：

  - **前端 (Frontend):**
    - **框架:** [Vue.js 3](https://v3.vuejs.org/) - 用于构建用户界面的渐进式 JavaScript 框架。
    - **样式:** [Tailwind CSS](https://tailwindcss.com/) - 一个功能类优先的 CSS 框架，用于快速构建定制化设计。
    - **组件库:** [daisyUI](https://daisyui.com/) - 一套基于 Tailwind CSS 的免费 UI 组件库，极大地提升了开发效率和界面美观度。
  - **后端 (Backend):**
    - **框架:** [Django](https://www.djangoproject.com/) - 一个高级 Python Web 框架，用于快速、安全地开发网站。
    - **API 框架:** [Django REST Framework (DRF)](https://www.django-rest-framework.org/) - 为 Django 提供强大的工具来构建 Web APIs。
    - **认证授权:** [JSON Web Token (JWT)](https://jwt.io/) - 用于实现安全的用户身份验证和会话管理。
    - **AI Agent 框架:** LangChain + [LangGraph](https://langchain-ai.github.io/langgraph/) - 用于构建有状态、容错的AI Agent，管理复杂的对话逻辑。
    - **RAG:** 通过自建知识库实现当需要时让Agent查询知识库，得到参考资料后作为系统提示词发给LLM使得LLM幻觉概率大大降低。
    - **智能体（Agent）技术:** 通过 LangChain 框架实现，使AI伙伴能够自主决策、调用工具（如知识库检索、时间查询等），从而执行更复杂的任务并提供更精准、动态的响应。
    - **数据库:** 
    - - [MySQL](https://www.mysql.com/) - 一个流行的开源关系型数据库管理系统，用于存储用户数据、对话历史和知识库元数据。

------

  ## ***\*核心功能\****

    1. **智能对话:** 用户可以与AI伙伴进行自然、流畅的多轮对话，体验接近真人般的交流感受。
    2. **个性化记忆:** AI Agent 能够学习和记住关于用户的独特信息（如姓名、兴趣、过往对话），形成个性化的长期记忆，使每次对话都更加连贯和贴心。
    3. **实时语音识别、语音合成、语音复刻:** 对话内容可以实时转换为语音流，提供沉浸式的听觉体验。同时也支持用户自定义音色。
    4. **AI生成/续写内容:** 可以使用AI生成/续写内容自动生成角色介绍,使得用户生成AI伙伴更加便捷
    5. **用户认证:** 使用JWT实现安全的用户注册、登录和API访问控制。
    6. **模块化扩展:** 基于LangGraph的Agent设计使得后续增加新的对话技能（如翻译、创作、编程等）变得易于实现。

------

  ## ***\*项目亮点\****

  - **技术前沿:** 整合了当前最热门的AI和Web开发技术，是学习和实践新技术的理想范例。 
  - **体验流畅:** 前端采用现代化UI框架，确保了界面的响应速度和交互的丝滑感。
  - **智能交互:** Agent技术的应用，使AI伙伴不仅能回答问题，还能自主调用工具获取信息、执行任务，有效解决了大模型幻觉问题，并使其能够回答特定领域或私有知识的问题，行为更加智能和动态。
  - **结构清晰:** 前后端分离，代码组织合理，便于团队协作和后期维护。
