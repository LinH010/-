# LeetCode 206 反转链表(链式存储) 2024/12/26 :smile:

## 1. 迭代写法

 ### 思想：

1. #### 维护指针pre和cur。

2. #### pre一开始指向空，cur指向头节点每次使得cur的next指向pre然后使pre和cur各向后走一步 
 ### 代码：
```java
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode * pre = NULL, * cur = head;
        while(cur){
            ListNode * t = cur->next;
            cur->next = pre;
            pre = cur;
            cur = t;
        }
        return pre;
    }
};
```

## 2. 递归写法

 ### 思想：	

1. #### 定义函数 ListNode* reverseList(ListNode * head): 返回以head为头结点的反转之后的链表的头结点

2. #### 要反转整个链表先反转以head->next为头结点的链表，再使得head.next.next = head, head.next = null

3. #### 特殊情况：head为空或head->next为空 返回head

   <img src="C:\Users\hasee\AppData\Roaming\Typora\typora-user-images\image-20241226161326765.png" alt="image-20241226161326765" style="zoom: 50%;" />
 ### 代码：
```java
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        if (!head || !head->next){
            return head;
        }
        ListNode* tail = reverseList(head->next);
        head->next->next = head;
        head->next = NULL;
        return tail;
    }
};
```

# LeetCode 15 三数之和(双指针) 2025/1/6 :smile:

 ### 思想：

1. #### 排序

2. #### 固定i向后找匹配的j与k (i < j < k)

3. #### 利用双指针算法删去不可能为答案的数 

   3.1 s > 0。 k -=1

   3.2 s < 0.    j +=1

   3.3 s == 0. ans里添加答案。

4. #### 去重。1.nums[i] == nums[i - 1] continue. 2. j,k同理
 ### 代码：
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        ans = []
        # 1.先排序
        nums.sort()
        # 2. 固定i 转化为两数之和  i,j,k
        for i in range(n-2):
            j = i + 1
            k = n - 1
            # 优化1 如果当前数与后两个数之和超过目标则无解
            if nums[i] + nums[i + 1] + nums[i + 2] > 0:
                break
            # 优化2 如果当前数与最大的两个数之和小于目标则无解
            if nums[i] + nums[n - 1] + nums[n - 2] < 0:
                continue
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            target = -nums[i]
            while j < k:
                s = nums[j] + nums[k]
                if s > target:
                    k-=1
                elif s < target:
                    j+=1
                else:
                    ans.append([nums[i],nums[j],nums[k]])
                    j+=1
                    while j < k and nums[j] == nums[j - 1]:
                        j+=1
                    k-=1
                    while j < k and nums[k] == nums[k + 1]:
                        k-=1
        return ans
```


# LeetCode 11 盛最多水的容器(双指针) 2025/1/6 :smile:

 ### 思想：

####  一句话：指针每一次移动，都意味着排除掉了一个柱子

1. #### 水的面积：宽度 * 高度 = $(j-i)\times\min(height[i],height[j])$

2. #### 要想水的面积最大那么宽度尽可能大，高度尽可能大

   #### 例子:

   ![image-20250106121550914](C:\Users\hasee\AppData\Roaming\Typora\typora-user-images\image-20250106121550914.png)

3. #### 利用双指针算法删去不可能为答案的数 

   ####  假定 i= 0,j = n - 1= 8 则面积 =宽　＊高 =８＊３＝２４

   3.1 固定较大的柱子7, 试图通过移动较小的柱子==更新==ans。由于已经有高度为7的柱子，==那么面积的高度不超过7==。有可能更新面积。==(宽度必然变少，但高度可能增加)==

   3.2 固定较小的柱子3, 试图通过移动较大的柱子==更新==ans。由于已经有高度为3的柱子，==那么面积的高度不超过3==。不可能更新面积。==(宽度必然变少，高度不可能增加)==

   3.3 综上，每次淘汰较小的柱子
 ### 代码：
```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        ans = 0
        n = len(height)
        i,j = 0,n - 1
        while i < j:
            ans = max(ans, (j - i) * min(height[i],height[j]))
            if height[i] < height[j]:
                i+=1
            elif height[i] > height[j]:
                j-=1
            else:
                i+=1
        return ans
```
